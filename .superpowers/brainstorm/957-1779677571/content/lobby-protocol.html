房间流程

从开始到游戏的完整流程
Step 1
房主点击「创建房间」→ 生成 PeerJS peer → 得到 4 位房间码（如 A3K9）
Step 2
房主在大厅界面配置：人类玩家数、AI 玩家数、规则设置
Step 3
其他玩家打开同一页面，输入房间码 → PeerJS 连接到房主 → 进入大厅
Step 4
所有人类玩家就位 → 房主点击「开始游戏」→ 游戏开始

消息协议

WebSocket 消息类型
客户端 → 房主
room:join { name }
game:play-card { cardId }
game:draw-card
game:pick-color { color }
game:accept-draw
game:resolve-uno { confirmed }
game:resolve-challenge { challenge }
房主 → 客户端
room:info { code, players, config }
game:state { 完整可见状态 }
game:your-turn { 提示轮到你 }
game:action-effect { 动画触发 }
game:deal-card { 逐张发牌动画 }
room:player-joined
room:player-left
